API Reference

Auth Endpoints

  • POST /api/core/auth/google
    Login with Google ID token
  • POST /api/core/auth/refresh
    Refresh access token
  • POST /api/core/auth/logout
    Logout and revoke session

Project1 Endpoints

  • GET /api/project1/notes
    Get all notes for current user
  • POST /api/project1/notes
    Create a new note (body: {title, body?})

Project2 Endpoints

  • GET /api/project2/tasks
    Get all tasks for current user
  • POST /api/project2/tasks
    Create a new task (body: {title, description?})
